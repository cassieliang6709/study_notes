# 项目

后端方向工程项目，涵盖分布式系统、AI/ML 基础设施和生产工具。

## 精选项目

- [OrderFlow](./orderflow/) — 基于 Java Spring Boot 的订单全生命周期管理后端
- [FinancialReport](./financial-report/) — 包含数据接入、检索、校验和报告组装的财务数据平台
- [MemoryOS](./memoryos/) — 基于 FastAPI、React、PostgreSQL 和 pgvector 的检索型产品系统
- [YouDescribe](./youdescribe/) — 生产实习：无障碍工具和 AI 辅助音频描述创作

## 其他项目

- **VisoCode** — 基于 LangGraph 的 multi-agent 系统，将自然语言转换为教学动画视频。在 AdventureX 2025 获得 Most Technically Outstanding。
- **AlgoMentor** — AI 编程辅导工具，设计了三层提示系统，防止用户过度依赖答案。
