# Python

这个模块负责补 Python OOP 的学习前置。它不是算法题区，也不是 OOD 案例区，而是帮助你在进入设计题前先把语言层面的基础补稳。

## 适合谁

- 看 `class`、`self`、`__init__` 容易乱的人
- 想看 OOD 文档，但总被 Python 语法卡住的人
- 已经会写函数，但对象思维还不稳的人

## 这里的文档怎么分工

- [Python OOP Prerequisites CN](./python_oop_prerequisites_cn.md)
  学 OOP 前必须会的 Python 基础
- [Python OOP Concepts CN](./python_oop_concepts_cn.md)
  类、对象、属性、方法、封装、继承、多态等核心概念

## 推荐阅读顺序

1. `Prerequisites`
2. `Concepts`
3. 再进入 [OOD](../ood/)
