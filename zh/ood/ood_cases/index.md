# OOD 案例

把 OOD 的抽象方法落到具体系统题。建议先看 [OOD 总览](../)，再按顺序看案例。

## 推荐顺序

1. [Parking Lot（停车场）](./parking_lot_python/) — 最基础的 OOD 入门题
2. [Amazon Locker](./amazon_locker_python/) — 状态机 + 多层结构
3. [Elevator System（电梯系统）](./elevator_system_python/) — 调度算法 + 并发状态

## 每篇案例重点看什么

- 系统范围怎么限定
- 核心对象有哪些
- 职责怎么拆分
- Python 代码怎么落地
- 后续扩展点怎么讲
