## 如果你准备按整本书来学

1. 先从 [Books](./books/) 选一本
2. `AI Projects Self Study`：先读 `How To Use This Book`，再选一条 `Track`
3. `Amazon VO Coding Book`：先读 `methodology`，再看 `high-frequency-list` 和 `problems`

## 读 coding 文档时建议的顺序

1. 看“这个题型 / 算法点的总结”
2. 看“题目含义”
3. 先自己判断方法
4. 再看 Python 代码
5. 看复杂度
6. 看“怎么想到”
7. 看示例 case 和常见 follow-up
