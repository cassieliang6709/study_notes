# Books

这里放的是独立成册、适合整本阅读的内容。它和根 README 的关系是：

- 根 README 负责全仓库导航
- `books` 负责某个主题的整本阅读体验

## 当前项目

- [AI Projects Self Study](./ai_projects_self_study/)
- [Amazon VO Coding Book](./amazon_vo_coding_book/book/)

## 每本书内部放什么

### AI Projects Self Study

- `How To Use This Book`：先建立正确学习方法
- `Track`：决定你按哪条主线学
- `Project Guide`：真正解释每个项目在教什么
- `7-Day Plan / Self Test / Answer Key`：负责练习和复盘

### Amazon VO Coding Book

- `methodology`：先理解这本书怎么还原题目、怎么准备 follow-up
- `high-frequency-list`：先定优先级
- `problems`：进入真正的题型还原和解法
- `documents / sources`：做资料索引和 mock 辅助

## 各自适合什么人

### AI Projects Self Study

适合按项目学 RAG、LLM from scratch、alignment、agent RL。

### Amazon VO Coding Book

适合按 Amazon 高频题单冲刺 coding interview。

## 推荐打开方式

### 如果你想学项目

1. [AI Projects Self Study](./ai_projects_self_study/)
2. `How To Use This Book`
3. 选一个 `Track`
4. 进入对应 `Project Guide`

### 如果你想冲 Amazon

1. [Amazon VO Coding Book](./amazon_vo_coding_book/book/)
2. `methodology`
3. `high-frequency-list`
4. `problems`
