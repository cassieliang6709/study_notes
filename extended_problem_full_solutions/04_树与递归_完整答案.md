## 1. Binary Tree Level Order Traversal

**题型识别**

```text
层序遍历 / Level Order BFS
```

**中文解释**

这是最标准的树 BFS 题。  
用队列一层一层处理，每次循环先记录当前层大小，然后把这一层所有节点弹出并把孩子加入队列。

**English Explanation**

This is the standard BFS traversal on a tree.  
Use a queue and process one level at a time by recording the current queue size, then pop all nodes in that level and push their children.

**Python 标准解法**

```python
from collections import deque
from typing import List, Optional


class Solution:
    def levelOrder(self, root: Optional['TreeNode']) -> List[List[int]]:
        if not root:
            return []

        ans = []
        q = deque([root])

        while q:
            level = []
            for _ in range(len(q)):
                node = q.popleft()
                level.append(node.val)
                if node.left:
                    q.append(node.left)
                if node.right:
                    q.append(node.right)
            ans.append(level)

        return ans
```

## 2. Binary Tree Right Side View

**题型识别**

```text
层序视图 / Level View
```

**中文解释**

右视图的意思是：每一层只看最右边那个节点。  
最直接做法是层序遍历，每层最后访问到的节点加入答案。

**English Explanation**

The right side view means: for each level, keep only the rightmost node.  
The most direct solution is level-order traversal and taking the last visited node in each level.

**Python 标准解法**

```python
from collections import deque
from typing import List, Optional


class Solution:
    def rightSideView(self, root: Optional['TreeNode']) -> List[int]:
        if not root:
            return []

        ans = []
        q = deque([root])

        while q:
            size = len(q)
            for i in range(size):
                node = q.popleft()
                if node.left:
                    q.append(node.left)
                if node.right:
                    q.append(node.right)
                if i == size - 1:
                    ans.append(node.val)

        return ans
```

## 3. Binary Tree Vertical Order Traversal

**题型识别**

```text
列号 BFS / BFS with Column Index
```

**中文解释**

给每个节点分配一个列号：

- 根节点列号是 `0`
- 左孩子是 `col - 1`
- 右孩子是 `col + 1`

然后按 BFS 顺序把节点值存到对应列中，最后按列号从小到大输出。

**English Explanation**

Assign each node a column index:

- root has column `0`
- left child is `col - 1`
- right child is `col + 1`

Then store node values by column during BFS, and finally output columns in sorted order.

**Python 标准解法**

```python
from collections import defaultdict, deque
from typing import List, Optional


class Solution:
    def verticalOrder(self, root: Optional['TreeNode']) -> List[List[int]]:
        if not root:
            return []

        cols = defaultdict(list)
        q = deque([(root, 0)])
        min_col = max_col = 0

        while q:
            node, col = q.popleft()
            cols[col].append(node.val)
            min_col = min(min_col, col)
            max_col = max(max_col, col)

            if node.left:
                q.append((node.left, col - 1))
            if node.right:
                q.append((node.right, col + 1))

        return [cols[c] for c in range(min_col, max_col + 1)]
```

## 4. Vertical Order Traversal of a Binary Tree

**题型识别**

```text
坐标排序 / Coordinate Sorting
```

**中文解释**

这题比上一题更严格，不仅要按列分组，还要考虑：

- 先按列
- 同列按行
- 如果行也相同，再按节点值排序

所以常见做法是 DFS 收集 `(col, row, value)`，最后排序。

**English Explanation**

This problem is stricter than simple vertical order:

- sort by column
- then by row
- if both are equal, sort by node value

So a common approach is to collect `(col, row, value)` tuples and sort them afterward.

**Python 标准解法**

```python
from collections import defaultdict
from typing import List, Optional


class Solution:
    def verticalTraversal(self, root: Optional['TreeNode']) -> List[List[int]]:
        cols = defaultdict(list)

        def dfs(node, row, col):
            if not node:
                return
            cols[col].append((row, node.val))
            dfs(node.left, row + 1, col - 1)
            dfs(node.right, row + 1, col + 1)

        dfs(root, 0, 0)

        ans = []
        for col in sorted(cols):
            ans.append([val for row, val in sorted(cols[col])])

        return ans
```

## 5. Find Largest Value in Each Tree Row

**题型识别**

```text
层序统计 / Level-wise Aggregation
```

**中文解释**

本质就是层序遍历，只不过每层不保存所有值，而是维护一个最大值。

**English Explanation**

This is simply level-order traversal, except we do not keep all values in a level; we only track the maximum value of that level.

**Python 标准解法**

```python
from collections import deque
from typing import List, Optional


class Solution:
    def largestValues(self, root: Optional['TreeNode']) -> List[int]:
        if not root:
            return []

        ans = []
        q = deque([root])

        while q:
            mx = float("-inf")
            for _ in range(len(q)):
                node = q.popleft()
                mx = max(mx, node.val)
                if node.left:
                    q.append(node.left)
                if node.right:
                    q.append(node.right)
            ans.append(mx)

        return ans
```

## 6. Maximum Width of Binary Tree

**题型识别**

```text
层序 + 位置编号 / Level Order with Indexing
```

**中文解释**

把树看成一棵完全二叉树，对每个节点编号：

- 左孩子 `2 * idx`
- 右孩子 `2 * idx + 1`

每层宽度 = 最右编号 - 最左编号 + 1。

**English Explanation**

Treat the tree like a complete binary tree and assign indices:

- left child `2 * idx`
- right child `2 * idx + 1`

Then the width of a level is:

```text
last_index - first_index + 1
```

**Python 标准解法**

```python
from collections import deque
from typing import Optional


class Solution:
    def widthOfBinaryTree(self, root: Optional['TreeNode']) -> int:
        q = deque([(root, 0)])
        ans = 0

        while q:
            _, first = q[0]
            _, last = q[-1]
            ans = max(ans, last - first + 1)

            for _ in range(len(q)):
                node, idx = q.popleft()
                if node.left:
                    q.append((node.left, 2 * idx))
                if node.right:
                    q.append((node.right, 2 * idx + 1))

        return ans
```

## 7. Populating Next Right Pointers in Each Node

**题型识别**

```text
层序链接 / Level Linking
```

**中文解释**

按层 BFS，把每层节点用 `next` 串起来。  
虽然这题是完美二叉树，也可以先用最稳的 BFS 写法。

**English Explanation**

Use level-order traversal and connect nodes in the same level using `next`.  
Although this is a perfect binary tree, BFS is the most straightforward and reliable approach.

**Python 标准解法**

```python
from collections import deque


class Solution:
    def connect(self, root: 'Optional[Node]') -> 'Optional[Node]':
        if not root:
            return root

        q = deque([root])

        while q:
            prev = None
            for _ in range(len(q)):
                node = q.popleft()
                if prev:
                    prev.next = node
                prev = node
                if node.left:
                    q.append(node.left)
                if node.right:
                    q.append(node.right)

        return root
```

## 9. Binary Tree Maximum Path Sum

**题型识别**

```text
后序 DFS + 全局答案 / Postorder DFS with Global Answer
```

**中文解释**

这题最容易混淆的点是：

- 返回给父节点的值
- 全局最大路径值

不是一回事。

对每个节点：

- 向父节点返回：`node.val + max(left_gain, right_gain)`
- 更新全局答案：`node.val + left_gain + right_gain`

负贡献要直接丢掉，所以要和 `0` 比较。

**English Explanation**

The key confusion in this problem is that:

- the value returned to the parent
- the global maximum path sum

are not the same.

For each node:

- return to parent: `node.val + max(left_gain, right_gain)`
- update global answer: `node.val + left_gain + right_gain`

Negative gains should be discarded, so compare them with `0`.

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def maxPathSum(self, root: Optional['TreeNode']) -> int:
        ans = float("-inf")

        def dfs(node):
            nonlocal ans
            if not node:
                return 0

            left = max(dfs(node.left), 0)
            right = max(dfs(node.right), 0)

            ans = max(ans, node.val + left + right)
            return node.val + max(left, right)

        dfs(root)
        return ans
```

## 10. Path Sum

**题型识别**

```text
树路径判断 / Root-to-Leaf Path Check
```

**中文解释**

从根往下减目标值。  
如果走到叶子节点时剩余值正好等于叶子值，就说明存在一条路径。

**English Explanation**

Subtract node values from the target as you go downward.  
If you reach a leaf and the remaining target equals the leaf value, then a valid path exists.

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def hasPathSum(self, root: Optional['TreeNode'], targetSum: int) -> bool:
        if not root:
            return False

        if not root.left and not root.right:
            return root.val == targetSum

        return (
            self.hasPathSum(root.left, targetSum - root.val)
            or self.hasPathSum(root.right, targetSum - root.val)
        )
```

## 11. Path Sum II

**题型识别**

```text
DFS 路径收集 / DFS Path Collection
```

**中文解释**

和上一题不同，这次不只是判断是否存在，而是要把所有合法路径收集出来。  
所以 DFS 时需要维护当前路径 `path`，到叶子节点时复制一份加入答案。

**English Explanation**

Unlike the previous problem, this one asks us to collect all valid paths instead of just checking existence.  
So during DFS we maintain the current path, and when reaching a valid leaf we append a copy of that path.

**Python 标准解法**

```python
from typing import List, Optional


class Solution:
    def pathSum(self, root: Optional['TreeNode'], targetSum: int) -> List[List[int]]:
        ans = []

        def dfs(node, remain, path):
            if not node:
                return

            path.append(node.val)

            if not node.left and not node.right and remain == node.val:
                ans.append(path[:])

            dfs(node.left, remain - node.val, path)
            dfs(node.right, remain - node.val, path)

            path.pop()

        dfs(root, targetSum, [])
        return ans
```

## 12. Path Sum III

**题型识别**

```text
树上前缀和 / Prefix Sum on Tree Paths
```

**中文解释**

这题和数组里的 `Subarray Sum Equals K` 很像。  
我们维护从根到当前节点的前缀和 `prefix`。  
如果之前出现过某个前缀和 `prefix - targetSum`，说明中间这一段路径和正好等于目标值。

**English Explanation**

This problem is very similar to `Subarray Sum Equals K` on arrays.  
We maintain the prefix sum from the root to the current node.  
If a previous prefix sum equals `prefix - targetSum`, then the path segment between them sums to the target.

**Python 标准解法**

```python
from collections import defaultdict
from typing import Optional


class Solution:
    def pathSum(self, root: Optional['TreeNode'], targetSum: int) -> int:
        count = defaultdict(int)
        count[0] = 1

        def dfs(node, prefix):
            if not node:
                return 0

            prefix += node.val
            ans = count[prefix - targetSum]
            count[prefix] += 1

            ans += dfs(node.left, prefix)
            ans += dfs(node.right, prefix)

            count[prefix] -= 1
            return ans

        return dfs(root, 0)
```

## 13. Sum Root to Leaf Numbers

**题型识别**

```text
路径数值累积 / Path Value Accumulation
```

**中文解释**

根到当前节点形成一个数字。  
每往下一层，相当于前面的数字乘 `10` 再加当前节点值。

**English Explanation**

Each root-to-current path forms a number.  
When moving one level down, multiply the current number by `10` and add the node value.

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def sumNumbers(self, root: Optional['TreeNode']) -> int:
        def dfs(node, cur):
            if not node:
                return 0

            cur = cur * 10 + node.val

            if not node.left and not node.right:
                return cur

            return dfs(node.left, cur) + dfs(node.right, cur)

        return dfs(root, 0)
```

## 14. Symmetric Tree

**题型识别**

```text
镜像递归 / Mirror Recursion
```

**中文解释**

对称树的关键不是分别判断左右子树，而是判断：

- 左树的左子树 vs 右树的右子树
- 左树的右子树 vs 右树的左子树

**English Explanation**

The key to symmetry is not checking each subtree independently, but comparing:

- left subtree's left child vs right subtree's right child
- left subtree's right child vs right subtree's left child

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def isSymmetric(self, root: Optional['TreeNode']) -> bool:
        def mirror(a, b):
            if not a and not b:
                return True
            if not a or not b or a.val != b.val:
                return False
            return mirror(a.left, b.right) and mirror(a.right, b.left)

        return mirror(root.left, root.right) if root else True
```

## 15. Diameter of Binary Tree

**题型识别**

```text
后序 DFS 求高度 / Postorder Height Computation
```

**中文解释**

对每个节点：

- 经过它的最长路径 = 左子树高度 + 右子树高度
- 返回给父节点的是自己的高度

所以这题是标准的“后序遍历 + 全局答案”。

**English Explanation**

For each node:

- the longest path passing through it = left height + right height
- the value returned to the parent is the node’s height

So this is a standard “postorder traversal + global answer” problem.

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def diameterOfBinaryTree(self, root: Optional['TreeNode']) -> int:
        ans = 0

        def depth(node):
            nonlocal ans
            if not node:
                return 0

            left = depth(node.left)
            right = depth(node.right)
            ans = max(ans, left + right)

            return 1 + max(left, right)

        depth(root)
        return ans
```

## 16. Flatten Binary Tree to Linked List

**题型识别**

```text
树结构重连 / Tree Rewiring
```

**中文解释**

目标是把树变成前序遍历顺序的链表。  
对当前节点：

1. 先拍平左右子树
2. 如果有左子树，就把左子树插到右边
3. 再把原右子树接到左子树尾部

**English Explanation**

The goal is to transform the tree into a linked list following preorder traversal order.  
For each node:

1. flatten left and right subtrees
2. if the left subtree exists, move it to the right side
3. attach the original right subtree to the tail of the moved left subtree

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def flatten(self, root: Optional['TreeNode']) -> None:
        def dfs(node):
            if not node:
                return None

            left_tail = dfs(node.left)
            right_tail = dfs(node.right)

            if left_tail:
                left_tail.right = node.right
                node.right = node.left
                node.left = None

            return right_tail or left_tail or node

        dfs(root)
```

## 17. Construct Binary Tree from Preorder and Inorder Traversal

**题型识别**

```text
递归构造 / Recursive Tree Construction
```

**中文解释**

关键性质：

- 前序第一个元素一定是根
- 中序中根左边是左子树，右边是右子树

所以每次递归：

1. 从前序取根
2. 在中序定位根
3. 递归构建左右子树

**English Explanation**

The key properties are:

- the first value in preorder is always the root
- in inorder, everything left of the root belongs to the left subtree, and everything right belongs to the right subtree

So recursively:

1. take the root from preorder
2. locate it in inorder
3. build left and right subtrees

**Python 标准解法**

```python
from typing import List, Optional


class Solution:
    def buildTree(self, preorder: List[int], inorder: List[int]) -> Optional['TreeNode']:
        pos = {value: i for i, value in enumerate(inorder)}
        pre_idx = 0

        def build(left, right):
            nonlocal pre_idx
            if left > right:
                return None

            root_val = preorder[pre_idx]
            pre_idx += 1

            root = TreeNode(root_val)
            mid = pos[root_val]

            root.left = build(left, mid - 1)
            root.right = build(mid + 1, right)

            return root

        return build(0, len(inorder) - 1)
```

## 18. Lowest Common Ancestor of a Binary Tree

**题型识别**

```text
后序递归判断 / Postorder Recursive Search
```

**中文解释**

如果：

- 左子树找到了一个目标
- 右子树也找到了另一个目标

那么当前节点就是最近公共祖先。  
否则，把找到的那个节点往上返回。

**English Explanation**

If:

- the left subtree finds one target
- the right subtree finds the other target

then the current node is the lowest common ancestor.  
Otherwise, return whichever side found a target.

**Python 标准解法**

```python
class Solution:
    def lowestCommonAncestor(self, root: 'TreeNode', p: 'TreeNode', q: 'TreeNode') -> 'TreeNode':
        if not root or root == p or root == q:
            return root

        left = self.lowestCommonAncestor(root.left, p, q)
        right = self.lowestCommonAncestor(root.right, p, q)

        if left and right:
            return root

        return left or right
```

## 19. Validate Binary Search Tree

**题型识别**

```text
BST 上下界检查 / BST Range Validation
```

**中文解释**

不能只检查：

```text
node.left < node < node.right
```

因为 BST 的约束是整棵子树范围。  
所以递归时要传入合法区间 `(low, high)`。

**English Explanation**

It is not enough to check only:

```text
node.left < node < node.right
```

because BST constraints apply to the entire subtree range.  
So we recursively pass down a valid interval `(low, high)`.

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def isValidBST(self, root: Optional['TreeNode']) -> bool:
        def valid(node, low, high):
            if not node:
                return True
            if not (low < node.val < high):
                return False
            return valid(node.left, low, node.val) and valid(node.right, node.val, high)

        return valid(root, float("-inf"), float("inf"))
```

## 20. Lowest Common Ancestor of a Binary Search Tree

**题型识别**

```text
BST 定向搜索 / Directed BST Search
```

**中文解释**

利用 BST 性质：

- 如果 `p` 和 `q` 都比根小，往左走
- 如果都比根大，往右走
- 否则当前根就是分叉点，也就是 LCA

**English Explanation**

Use the BST property:

- if both `p` and `q` are smaller than the root, go left
- if both are larger, go right
- otherwise, the current root is the split point, so it is the LCA

**Python 标准解法**

```python
class Solution:
    def lowestCommonAncestor(self, root: 'TreeNode', p: 'TreeNode', q: 'TreeNode') -> 'TreeNode':
        while root:
            if p.val < root.val and q.val < root.val:
                root = root.left
            elif p.val > root.val and q.val > root.val:
                root = root.right
            else:
                return root
```

## 21. Convert Sorted Array to Binary Search Tree

**题型识别**

```text
有序数组建平衡 BST / Build Balanced BST
```

**中文解释**

为了尽量平衡，根节点应选中间元素。  
左右两边继续递归构建左右子树。

**English Explanation**

To keep the BST balanced, choose the middle element as the root.  
Then recursively build the left and right subtrees from the two halves.

**Python 标准解法**

```python
from typing import List, Optional


class Solution:
    def sortedArrayToBST(self, nums: List[int]) -> Optional['TreeNode']:
        def build(left, right):
            if left > right:
                return None
            mid = (left + right) // 2
            root = TreeNode(nums[mid])
            root.left = build(left, mid - 1)
            root.right = build(mid + 1, right)
            return root

        return build(0, len(nums) - 1)
```

## 22. Convert Binary Search Tree to Sorted Doubly Linked List

**题型识别**

```text
BST 中序重连 / Inorder Rewiring
```

**中文解释**

BST 的中序遍历天然有序。  
所以中序遍历过程中，维护：

- `first`：链表头
- `last`：前一个访问到的节点

每访问一个节点，就把它和 `last` 连起来。  
最后再把头尾首尾相连。

**English Explanation**

Inorder traversal of a BST is naturally sorted.  
So during inorder traversal, maintain:

- `first`: head of the doubly linked list
- `last`: previously visited node

Each visited node is connected to `last`, and finally the head and tail are linked together.

**Python 标准解法**

```python
class Solution:
    def treeToDoublyList(self, root: 'Optional[Node]') -> 'Optional[Node]':
        if not root:
            return None

        first = last = None

        def dfs(node):
            nonlocal first, last
            if not node:
                return

            dfs(node.left)

            if last:
                last.right = node
                node.left = last
            else:
                first = node
            last = node

            dfs(node.right)

        dfs(root)
        first.left = last
        last.right = first
        return first
```

## 23. Kth Smallest Element in a BST

**题型识别**

```text
BST 中序遍历 / Inorder Traversal of BST
```

**中文解释**

BST 中序遍历结果是递增的。  
所以只要中序遍历到第 `k` 个节点，返回它的值即可。

**English Explanation**

The inorder traversal of a BST is sorted in ascending order.  
So we simply perform inorder traversal and return the value of the k-th visited node.

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def kthSmallest(self, root: Optional['TreeNode'], k: int) -> int:
        stack = []

        while True:
            while root:
                stack.append(root)
                root = root.left

            root = stack.pop()
            k -= 1
            if k == 0:
                return root.val

            root = root.right
```

## 24. Range Sum of BST

**题型识别**

```text
BST 剪枝 DFS / BST Pruning DFS
```

**中文解释**

利用 BST 性质剪枝：

- 如果当前值小于 `low`，左子树都不用看
- 如果当前值大于 `high`，右子树都不用看

**English Explanation**

Use BST pruning:

- if current value is less than `low`, skip the entire left subtree
- if current value is greater than `high`, skip the entire right subtree

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def rangeSumBST(self, root: Optional['TreeNode'], low: int, high: int) -> int:
        if not root:
            return 0
        if root.val < low:
            return self.rangeSumBST(root.right, low, high)
        if root.val > high:
            return self.rangeSumBST(root.left, low, high)
        return (
            root.val
            + self.rangeSumBST(root.left, low, high)
            + self.rangeSumBST(root.right, low, high)
        )
```

## 25. Serialize and Deserialize Binary Tree

**题型识别**

```text
前序序列化 / Preorder Serialization
```

**中文解释**

最稳的做法是前序遍历 + 空节点标记。  
例如用 `#` 表示空节点。  
这样反序列化时按同样顺序递归重建即可。

**English Explanation**

The most reliable approach is preorder traversal with null markers.  
For example, use `#` for missing nodes.  
Then deserialize recursively in the same order.

**Python 标准解法**

```python
from collections import deque


class Codec:
    def serialize(self, root):
        vals = []

        def dfs(node):
            if not node:
                vals.append("#")
                return
            vals.append(str(node.val))
            dfs(node.left)
            dfs(node.right)

        dfs(root)
        return ",".join(vals)

    def deserialize(self, data):
        vals = deque(data.split(","))

        def dfs():
            val = vals.popleft()
            if val == "#":
                return None
            node = TreeNode(int(val))
            node.left = dfs()
            node.right = dfs()
            return node

        return dfs()
```
