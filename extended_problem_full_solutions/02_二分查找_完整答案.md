## 1. Search in Rotated Sorted Array

**题型识别**

```text
旋转数组二分 / Binary Search on Rotated Array
```

**中文解释**

虽然数组被旋转了，但每次二分后，至少有一半区间是有序的。  
我们先判断左半边是否有序：

- 如果左半边有序，再判断 `target` 是否落在这个有序区间内
- 否则说明右半边有序，再判断 `target` 是否落在右半边

**English Explanation**

Even though the array is rotated, after each binary split, at least one half is still sorted.  
We first check whether the left half is sorted:

- If the left half is sorted, check whether `target` falls inside it
- Otherwise, the right half must be sorted, so check that side

**Python 标准解法**

```python
from typing import List


class Solution:
    def search(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1

        while left <= right:
            mid = (left + right) // 2

            if nums[mid] == target:
                return mid

            if nums[left] <= nums[mid]:
                if nums[left] <= target < nums[mid]:
                    right = mid - 1
                else:
                    left = mid + 1
            else:
                if nums[mid] < target <= nums[right]:
                    left = mid + 1
                else:
                    right = mid - 1

        return -1
```

## 2. Find First and Last Position of Element in Sorted Array

**题型识别**

```text
边界二分 / Boundary Binary Search
```

**中文解释**

这题不是普通“找一个值”，而是找：

- 第一个等于 `target` 的位置
- 最后一个等于 `target` 的位置

所以要做两次二分：

1. 找左边界
2. 找 `target + 1` 的左边界，再减一得到右边界

**English Explanation**

This is not a normal “find one target” problem.  
We need:

- the first position equal to `target`
- the last position equal to `target`

So we run binary search twice:

1. Find the left boundary of `target`
2. Find the left boundary of `target + 1`, then subtract one

**Python 标准解法**

```python
from typing import List


class Solution:
    def searchRange(self, nums: List[int], target: int) -> List[int]:
        def lower_bound(x: int) -> int:
            left, right = 0, len(nums)
            while left < right:
                mid = (left + right) // 2
                if nums[mid] < x:
                    left = mid + 1
                else:
                    right = mid
            return left

        left = lower_bound(target)
        right = lower_bound(target + 1) - 1

        if left == len(nums) or nums[left] != target:
            return [-1, -1]

        return [left, right]
```

## 3. Find Peak Element

**题型识别**

```text
单调趋势二分 / Binary Search on Slope
```

**中文解释**

如果 `nums[mid] < nums[mid + 1]`，说明右边是上坡，峰值一定在右边。  
如果 `nums[mid] > nums[mid + 1]`，说明峰值在左边或当前点。  
所以可以通过比较相邻元素，逐步缩小区间。

**English Explanation**

If `nums[mid] < nums[mid + 1]`, then we are on an upward slope, so a peak must exist on the right.  
If `nums[mid] > nums[mid + 1]`, then a peak lies on the left side or at `mid`.  
So we shrink the interval using local slope information.

**Python 标准解法**

```python
from typing import List


class Solution:
    def findPeakElement(self, nums: List[int]) -> int:
        left, right = 0, len(nums) - 1

        while left < right:
            mid = (left + right) // 2
            if nums[mid] < nums[mid + 1]:
                left = mid + 1
            else:
                right = mid

        return left
```

## 4. Single Element in a Sorted Array

**题型识别**

```text
奇偶规律二分 / Binary Search with Pair Parity
```

**中文解释**

在有序数组里，如果每个元素都成对出现，那么：

- 第一个元素对会从偶数下标开始
- 第二个元素对也会从偶数下标开始

一旦唯一元素出现，这个规律就被打破。  
所以我们让 `mid` 落在偶数位，然后检查：

- 如果 `nums[mid] == nums[mid + 1]`，唯一元素在右边
- 否则在左边或就是 `mid`

**English Explanation**

In a sorted array where every element appears twice, each pair starts at an even index.  
Once the single element appears, this pattern breaks.  
So we force `mid` to be even and check:

- If `nums[mid] == nums[mid + 1]`, the single element is on the right
- Otherwise, it is on the left side or at `mid`

**Python 标准解法**

```python
from typing import List


class Solution:
    def singleNonDuplicate(self, nums: List[int]) -> int:
        left, right = 0, len(nums) - 1

        while left < right:
            mid = (left + right) // 2
            if mid % 2 == 1:
                mid -= 1

            if nums[mid] == nums[mid + 1]:
                left = mid + 2
            else:
                right = mid

        return nums[left]
```

## 5. Search a 2D Matrix

**题型识别**

```text
二维转一维二分 / 2D as 1D Binary Search
```

**中文解释**

因为题目满足：

- 每一行有序
- 每一行的第一个数大于上一行最后一个数

所以整个矩阵其实相当于一个升序一维数组。  
只要把一维下标 `mid` 映射回二维：

- 行号：`mid // n`
- 列号：`mid % n`

**English Explanation**

Since:

- each row is sorted
- the first number of each row is greater than the last number of the previous row

the whole matrix behaves like a single sorted 1D array.  
We map a 1D index `mid` back to 2D:

- row = `mid // n`
- col = `mid % n`

**Python 标准解法**

```python
from typing import List


class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        if not matrix or not matrix[0]:
            return False

        m, n = len(matrix), len(matrix[0])
        left, right = 0, m * n - 1

        while left <= right:
            mid = (left + right) // 2
            value = matrix[mid // n][mid % n]

            if value == target:
                return True
            if value < target:
                left = mid + 1
            else:
                right = mid - 1

        return False
```

## 6. Find Minimum in Rotated Sorted Array

**题型识别**

```text
旋转数组最小值二分 / Rotated Array Minimum
```

**中文解释**

比较 `nums[mid]` 和 `nums[right]`：

- 如果 `nums[mid] > nums[right]`，最小值一定在右边
- 否则最小值在左边，包括 `mid`

这题的本质是：最小值所在位置把数组分成了两段有序区间。

**English Explanation**

Compare `nums[mid]` with `nums[right]`:

- If `nums[mid] > nums[right]`, the minimum must be on the right side
- Otherwise, the minimum is on the left side, including `mid`

The key idea is that the minimum splits the rotated array into two sorted parts.

**Python 标准解法**

```python
from typing import List


class Solution:
    def findMin(self, nums: List[int]) -> int:
        left, right = 0, len(nums) - 1

        while left < right:
            mid = (left + right) // 2
            if nums[mid] > nums[right]:
                left = mid + 1
            else:
                right = mid

        return nums[left]
```

## 7. Search in Rotated Sorted Array II

**题型识别**

```text
带重复的旋转数组二分 / Rotated Binary Search with Duplicates
```

**中文解释**

和前面的旋转数组搜索类似，但这里有重复值。  
当出现：

```text
nums[left] == nums[mid] == nums[right]
```

时，你无法判断哪一边有序，只能把边界缩掉一格继续找。

**English Explanation**

This is similar to rotated binary search, but duplicates are allowed.  
When:

```text
nums[left] == nums[mid] == nums[right]
```

we cannot determine which half is sorted, so we shrink both boundaries by one.

**Python 标准解法**

```python
from typing import List


class Solution:
    def search(self, nums: List[int], target: int) -> bool:
        left, right = 0, len(nums) - 1

        while left <= right:
            mid = (left + right) // 2

            if nums[mid] == target:
                return True

            if nums[left] == nums[mid] == nums[right]:
                left += 1
                right -= 1
            elif nums[left] <= nums[mid]:
                if nums[left] <= target < nums[mid]:
                    right = mid - 1
                else:
                    left = mid + 1
            else:
                if nums[mid] < target <= nums[right]:
                    left = mid + 1
                else:
                    right = mid - 1

        return False
```

## 8. Search a 2D Matrix II

**题型识别**

```text
矩阵单调搜索 / Monotonic Matrix Search
```

**中文解释**

这题虽然也在二分类里，但标准做法不是普通二分，而是从右上角开始搜：

- 如果当前值太大，往左走
- 如果当前值太小，往下走

因为：

- 左边更小
- 下边更大

**English Explanation**

Although this is often grouped with binary search, the standard solution is to start from the top-right corner:

- If the current value is too large, move left
- If it is too small, move down

because:

- values to the left are smaller
- values below are larger

**Python 标准解法**

```python
from typing import List


class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        if not matrix or not matrix[0]:
            return False

        row, col = 0, len(matrix[0]) - 1

        while row < len(matrix) and col >= 0:
            if matrix[row][col] == target:
                return True
            if matrix[row][col] > target:
                col -= 1
            else:
                row += 1

        return False
```

## 9. Random Pick with Weight

**题型识别**

```text
前缀和 + 二分 / Prefix Sum + Binary Search
```

**中文解释**

如果权重是 `[1, 3, 2]`，那么前缀和是 `[1, 4, 6]`。  
随机生成 `1..6` 之间的一个数，看它落在哪个前缀区间里：

- `1` -> 下标 0
- `2,3,4` -> 下标 1
- `5,6` -> 下标 2

所以本质是：

1. 建前缀和
2. 随机取一个数
3. 二分找第一个 `>= x` 的位置

**English Explanation**

If weights are `[1, 3, 2]`, then prefix sums are `[1, 4, 6]`.  
Generate a random number from `1..6` and find which prefix interval contains it:

- `1` -> index 0
- `2,3,4` -> index 1
- `5,6` -> index 2

So the method is:

1. Build prefix sums
2. Draw a random number
3. Binary search the first prefix sum `>= x`

**Python 标准解法**

```python
from bisect import bisect_left
from random import randint
from typing import List


class Solution:
    def __init__(self, w: List[int]):
        self.prefix = []
        total = 0
        for x in w:
            total += x
            self.prefix.append(total)
        self.total = total

    def pickIndex(self) -> int:
        x = randint(1, self.total)
        return bisect_left(self.prefix, x)
```

## 10. Kth Smallest Element in a Sorted Matrix

**题型识别**

```text
值域二分 / Binary Search on Value Space
```

**中文解释**

这题容易误以为是对下标二分，其实不是。  
我们在“值的范围”上二分：

- `left = matrix[0][0]`
- `right = matrix[-1][-1]`

对于某个 `mid`，统计矩阵里有多少个数 `<= mid`：

- 如果数量小于 `k`，说明第 `k` 小更大
- 否则说明答案不大于 `mid`

**English Explanation**

This problem is often mistaken as index-based binary search, but it is really binary search on the value range:

- `left = matrix[0][0]`
- `right = matrix[-1][-1]`

For a given `mid`, count how many values in the matrix are `<= mid`:

- If the count is less than `k`, the k-th smallest must be larger
- Otherwise, the answer is at most `mid`

**Python 标准解法**

```python
from typing import List


class Solution:
    def kthSmallest(self, matrix: List[List[int]], k: int) -> int:
        n = len(matrix)

        def count_le(x: int) -> int:
            count = 0
            row, col = n - 1, 0
            while row >= 0 and col < n:
                if matrix[row][col] <= x:
                    count += row + 1
                    col += 1
                else:
                    row -= 1
            return count

        left, right = matrix[0][0], matrix[-1][-1]

        while left < right:
            mid = (left + right) // 2
            if count_le(mid) < k:
                left = mid + 1
            else:
                right = mid

        return left
```

## 11. Median of Two Sorted Arrays

**题型识别**

```text
划分二分 / Partition Binary Search
```

**中文解释**

这题核心不是“合并两个数组”，而是“找一个合法划分”。

设两个数组是 `A` 和 `B`，我们想把它们划成左右两半，满足：

1. 左半边元素总数和右半边平衡
2. 左半边最大值 `<=` 右半边最小值

只需要在较短数组上二分划分点 `i`，另一个数组的划分点 `j` 就能确定。  
一旦满足边界关系，答案就能直接算出来。

**English Explanation**

The key is not to merge the arrays, but to find a valid partition.

Suppose the arrays are `A` and `B`. We want to split them into left and right halves such that:

1. the two halves are balanced in size
2. the maximum on the left is `<=` the minimum on the right

We only binary search the partition index `i` in the shorter array.  
Then the partition index `j` in the other array is determined automatically.  
Once the partition is valid, the median can be computed directly.

**Python 标准解法**

```python
from typing import List


class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        A, B = nums1, nums2
        if len(A) > len(B):
            A, B = B, A

        total = len(A) + len(B)
        half = total // 2
        left, right = 0, len(A)

        while True:
            i = (left + right) // 2
            j = half - i

            Aleft = A[i - 1] if i > 0 else float("-inf")
            Aright = A[i] if i < len(A) else float("inf")
            Bleft = B[j - 1] if j > 0 else float("-inf")
            Bright = B[j] if j < len(B) else float("inf")

            if Aleft <= Bright and Bleft <= Aright:
                if total % 2:
                    return min(Aright, Bright)
                return (max(Aleft, Bleft) + min(Aright, Bright)) / 2
            elif Aleft > Bright:
                right = i - 1
            else:
                left = i + 1
```

## 12. Closest Binary Search Tree Value

**题型识别**

```text
BST 定向搜索 / Directed BST Search
```

**中文解释**

利用 BST 的性质：

- 如果 `target < root.val`，更接近的值更可能在左子树
- 否则更可能在右子树

一路向下，同时维护当前最接近的值即可。

**English Explanation**

Use the BST property:

- If `target < root.val`, a closer value is more likely on the left
- Otherwise, it is more likely on the right

Walk downward and keep track of the closest value seen so far.

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def closestValue(self, root: Optional['TreeNode'], target: float) -> int:
        closest = root.val

        while root:
            if abs(root.val - target) < abs(closest - target):
                closest = root.val

            if target < root.val:
                root = root.left
            else:
                root = root.right

        return closest
```

## 13. Find the Celebrity

**题型识别**

```text
候选人消元 / Candidate Elimination
```

**中文解释**

名人的定义是：

- 所有人都认识他
- 他不认识任何人

先遍历一遍做候选人筛选：

- 如果 `cand` 认识 `i`，那 `cand` 不可能是名人，更新候选人为 `i`

这样最后只剩下一个可能候选人。  
然后再做一次验证，检查：

- 候选人是否认识别人
- 别人是否都认识候选人

**English Explanation**

A celebrity is defined as:

- everyone knows them
- they know nobody else

First, eliminate candidates in one pass:

- If `cand` knows `i`, then `cand` cannot be the celebrity, so replace `cand` with `i`

After this pass, only one possible candidate remains.  
Then verify:

- the candidate knows nobody
- everybody knows the candidate

**Python 标准解法**

```python
# The knows API is already defined for you.


class Solution:
    def findCelebrity(self, n: int) -> int:
        cand = 0

        for i in range(1, n):
            if knows(cand, i):
                cand = i

        for i in range(n):
            if i == cand:
                continue
            if knows(cand, i) or not knows(i, cand):
                return -1

        return cand
```
