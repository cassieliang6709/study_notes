## 1. Reverse Linked List

**题型识别**

```text
链表反转模板 / Linked List Reversal
```

**中文解释**

反转链表的标准做法是维护两个指针：

- `prev`：已经反转好的部分头节点
- `cur`：当前要处理的节点

每一步都先保存 `next`，再把当前节点指向 `prev`。

**English Explanation**

The standard linked list reversal keeps two pointers:

- `prev`: the head of the already reversed part
- `cur`: the current node being processed

At each step, save `next` first, then point the current node back to `prev`.

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def reverseList(self, head: Optional['ListNode']) -> Optional['ListNode']:
        prev = None
        cur = head

        while cur:
            nxt = cur.next
            cur.next = prev
            prev = cur
            cur = nxt

        return prev
```

## 2. Reverse Linked List II

**题型识别**

```text
局部链表反转 / Partial Linked List Reversal
```

**中文解释**

这题是“只反转一段区间”。  
关键是先找到反转区间前一个节点 `prev`，然后对区间做头插法重排。

为什么要用 `dummy`：

- 如果反转区间从头节点开始，`dummy` 可以统一处理

**English Explanation**

This problem asks us to reverse only a sublist.  
The key is to first find the node before the reversal segment (`prev`), then perform head-insertion inside the segment.

Why use a `dummy` node:

- If the segment starts at the head, `dummy` lets us handle it uniformly

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def reverseBetween(self, head: Optional['ListNode'], left: int, right: int) -> Optional['ListNode']:
        dummy = ListNode(0, head)
        prev = dummy

        for _ in range(left - 1):
            prev = prev.next

        cur = prev.next
        for _ in range(right - left):
            nxt = cur.next
            cur.next = nxt.next
            nxt.next = prev.next
            prev.next = nxt

        return dummy.next
```

## 3. Reorder List

**题型识别**

```text
找中点 + 反转 + 合并 / Split + Reverse + Merge
```

**中文解释**

题目要求把链表重排成：

```text
L0 -> Ln -> L1 -> Ln-1 -> ...
```

标准做法分三步：

1. 快慢指针找中点
2. 反转后半段
3. 前后两段交替合并

**English Explanation**

We need to reorder the list into:

```text
L0 -> Ln -> L1 -> Ln-1 -> ...
```

The standard approach has three steps:

1. find the middle using slow/fast pointers
2. reverse the second half
3. merge the two halves alternately

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def reorderList(self, head: Optional['ListNode']) -> None:
        if not head or not head.next:
            return

        slow = fast = head
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next

        prev, cur = None, slow.next
        slow.next = None
        while cur:
            nxt = cur.next
            cur.next = prev
            prev = cur
            cur = nxt

        first, second = head, prev
        while second:
            n1, n2 = first.next, second.next
            first.next = second
            second.next = n1
            first, second = n1, n2
```

## 4. Remove Nth Node From End of List

**题型识别**

```text
快慢指针 / Two Pointers
```

**中文解释**

让 `fast` 先走 `n` 步，然后 `slow` 和 `fast` 一起走。  
这样当 `fast` 到达末尾时，`slow` 正好在待删除节点前一个位置。

为什么要用 `dummy`：

- 如果删除的是头节点，`dummy` 能统一处理

**English Explanation**

Move `fast` ahead by `n` steps, then move `slow` and `fast` together.  
When `fast` reaches the end, `slow` will be exactly one node before the node to delete.

Why use `dummy`:

- If the head must be removed, `dummy` handles it cleanly

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def removeNthFromEnd(self, head: Optional['ListNode'], n: int) -> Optional['ListNode']:
        dummy = ListNode(0, head)
        fast = slow = dummy

        for _ in range(n):
            fast = fast.next

        while fast.next:
            fast = fast.next
            slow = slow.next

        slow.next = slow.next.next
        return dummy.next
```

## 5. Add Two Numbers

**题型识别**

```text
链表模拟加法 / Linked List Addition
```

**中文解释**

这题和手算加法完全一样：

- 当前位 = 两个节点值 + 进位
- 结果节点保存 `total % 10`
- 新进位是 `total // 10`

因为输入本身是逆序，所以我们直接从头往后加。

**English Explanation**

This is exactly like manual addition:

- current digit = two node values + carry
- store `total % 10`
- new carry = `total // 10`

Since the input is already in reverse order, we can add directly from head to tail.

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def addTwoNumbers(self, l1: Optional['ListNode'], l2: Optional['ListNode']) -> Optional['ListNode']:
        dummy = ListNode()
        cur = dummy
        carry = 0

        while l1 or l2 or carry:
            x = l1.val if l1 else 0
            y = l2.val if l2 else 0
            total = x + y + carry

            carry = total // 10
            cur.next = ListNode(total % 10)
            cur = cur.next

            l1 = l1.next if l1 else None
            l2 = l2.next if l2 else None

        return dummy.next
```

## 6. Add Two Numbers II

**题型识别**

```text
栈 + 链表加法 / Stack-Based Linked List Addition
```

**中文解释**

这次输入是正序的，不能像上一题那样直接从头加。  
最稳的做法是：

1. 把两个链表值压栈
2. 从栈顶开始相加
3. 每次把新节点插到结果链表头部

**English Explanation**

Now the digits are stored in forward order, so we cannot add directly from the head.  
The clean approach is:

1. push all digits into stacks
2. add from the stack tops
3. insert each new node at the head of the result list

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def addTwoNumbers(self, l1: Optional['ListNode'], l2: Optional['ListNode']) -> Optional['ListNode']:
        s1, s2 = [], []

        while l1:
            s1.append(l1.val)
            l1 = l1.next
        while l2:
            s2.append(l2.val)
            l2 = l2.next

        carry = 0
        head = None

        while s1 or s2 or carry:
            total = carry
            if s1:
                total += s1.pop()
            if s2:
                total += s2.pop()

            node = ListNode(total % 10)
            node.next = head
            head = node
            carry = total // 10

        return head
```

## 7. Merge Two Sorted Lists

**题型识别**

```text
有序链表合并 / Merge Two Sorted Lists
```

**中文解释**

这题是链表里的最基础模板题。  
维护一个 `tail` 指针，每次把较小节点接到结果链表后面。

**English Explanation**

This is one of the most basic linked-list templates.  
Maintain a `tail` pointer and always append the smaller current node.

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def mergeTwoLists(self, list1: Optional['ListNode'], list2: Optional['ListNode']) -> Optional['ListNode']:
        dummy = ListNode()
        tail = dummy

        while list1 and list2:
            if list1.val <= list2.val:
                tail.next = list1
                list1 = list1.next
            else:
                tail.next = list2
                list2 = list2.next
            tail = tail.next

        tail.next = list1 or list2
        return dummy.next
```

## 8. Merge k Sorted Lists

**题型识别**

```text
堆 / Min Heap
```

**中文解释**

合并两个有序链表可以双指针，但合并 `k` 个时最自然的是堆。  
堆里维护每条链表当前头节点：

- 每次弹出最小节点接到答案后面
- 再把它的 `next` 放回堆中

**English Explanation**

Merging two sorted lists can be done with two pointers, but for `k` lists the natural solution is a heap.  
The heap stores the current head of each list:

- pop the smallest node and append it
- then push its `next` node back into the heap

**Python 标准解法**

```python
import heapq
from typing import List, Optional


class Solution:
    def mergeKLists(self, lists: List[Optional['ListNode']]) -> Optional['ListNode']:
        heap = []

        for i, node in enumerate(lists):
            if node:
                heapq.heappush(heap, (node.val, i, node))

        dummy = ListNode()
        tail = dummy

        while heap:
            _, i, node = heapq.heappop(heap)
            tail.next = node
            tail = tail.next

            if node.next:
                heapq.heappush(heap, (node.next.val, i, node.next))

        return dummy.next
```

## 9. Sort List

**题型识别**

```text
链表归并排序 / Merge Sort on Linked List
```

**中文解释**

链表不适合快速排序，因为随机访问不方便。  
归并排序更自然：

1. 找中点拆成两半
2. 递归排序左右两半
3. 再合并两个有序链表

**English Explanation**

Quick sort is not natural for linked lists because random access is inconvenient.  
Merge sort is a better fit:

1. split into two halves using the middle
2. recursively sort both halves
3. merge the two sorted lists

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def sortList(self, head: Optional['ListNode']) -> Optional['ListNode']:
        if not head or not head.next:
            return head

        slow, fast = head, head.next
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next

        mid = slow.next
        slow.next = None

        left = self.sortList(head)
        right = self.sortList(mid)
        return self.merge(left, right)

    def merge(self, a: Optional['ListNode'], b: Optional['ListNode']) -> Optional['ListNode']:
        dummy = ListNode()
        tail = dummy

        while a and b:
            if a.val <= b.val:
                tail.next = a
                a = a.next
            else:
                tail.next = b
                b = b.next
            tail = tail.next

        tail.next = a or b
        return dummy.next
```

## 10. Intersection of Two Linked Lists

**题型识别**

```text
双指针路径对齐 / Two Pointers with Path Alignment
```

**中文解释**

让两个指针分别走：

- `A -> B`
- `B -> A`

这样走完以后，两者总路程相同，就会在交点相遇；如果没有交点，就一起走到 `None`。

**English Explanation**

Let the two pointers traverse:

- `A -> B`
- `B -> A`

This equalizes the total distance traveled, so they meet at the intersection node; if there is no intersection, they both reach `None`.

**Python 标准解法**

```python
from typing import Optional


class Solution:
    def getIntersectionNode(self, headA: 'ListNode', headB: 'ListNode') -> Optional['ListNode']:
        a, b = headA, headB

        while a is not b:
            a = a.next if a else headB
            b = b.next if b else headA

        return a
```

## 11. Copy List with Random Pointer

**题型识别**

```text
哈希映射复制 / Hash Map Copy
```

**中文解释**

这题难点在于除了 `next` 之外还有 `random` 指针。  
最稳的写法是：

1. 第一遍创建所有新节点，建立 `旧节点 -> 新节点` 的映射
2. 第二遍连接 `next` 和 `random`

**English Explanation**

The difficulty here is the extra `random` pointer in addition to `next`.  
The cleanest approach is:

1. first pass: create all copied nodes and build a map `old -> new`
2. second pass: connect `next` and `random`

**Python 标准解法**

```python
class Solution:
    def copyRandomList(self, head: 'Optional[Node]') -> 'Optional[Node]':
        if not head:
            return None

        old_to_new = {}
        cur = head

        while cur:
            old_to_new[cur] = Node(cur.val)
            cur = cur.next

        cur = head
        while cur:
            old_to_new[cur].next = old_to_new.get(cur.next)
            old_to_new[cur].random = old_to_new.get(cur.random)
            cur = cur.next

        return old_to_new[head]
```
