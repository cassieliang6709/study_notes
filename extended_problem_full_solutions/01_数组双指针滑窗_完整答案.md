## 1. Two Sum

**题型识别**

```text
哈希表 / Hash Map
```

**中文解释**

题目问的是：是否存在两个数之和等于 `target`。  
扫到当前数字 `x` 时，只需要知道之前有没有出现过 `target - x`。  
所以用字典记录“某个值第一次出现的位置”。

**English Explanation**

The problem asks whether there exist two numbers whose sum is `target`.  
When we are at the current number `x`, we only need to know whether `target - x` has appeared before.  
So we use a hash map to store `value -> index`.

**Python 标准解法**

```python
from typing import List


class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        pos = {}

        for i, x in enumerate(nums):
            need = target - x
            if need in pos:
                return [pos[need], i]
            pos[x] = i

        return []
```

## 2. Product of Array Except Self

**题型识别**

```text
前缀乘积 + 后缀乘积 / Prefix and Suffix Products
```

**中文解释**

`answer[i]` 要等于 `i` 左边所有数的乘积乘上右边所有数的乘积。  
第一遍从左到右，把左边乘积存进 `answer`。  
第二遍从右到左，再乘上右边乘积。

**English Explanation**

`answer[i]` should be the product of all numbers to the left of `i` times the product of all numbers to the right of `i`.  
In the first pass, we store left products.  
In the second pass, we multiply by right products.

**Python 标准解法**

```python
from typing import List


class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        n = len(nums)
        ans = [1] * n

        left = 1
        for i in range(n):
            ans[i] = left
            left *= nums[i]

        right = 1
        for i in range(n - 1, -1, -1):
            ans[i] *= right
            right *= nums[i]

        return ans
```

## 3. Move Zeroes

**题型识别**

```text
快慢指针 / Slow-Fast Pointers
```

**中文解释**

`slow` 表示下一个非零元素应该放的位置，`fast` 负责扫描数组。  
每当 `fast` 遇到非零元素，就把它交换到 `slow` 位置。

**English Explanation**

`slow` points to the next position where a non-zero number should go, while `fast` scans the array.  
Whenever `fast` sees a non-zero value, we swap it into position `slow`.

**Python 标准解法**

```python
from typing import List


class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        slow = 0

        for fast in range(len(nums)):
            if nums[fast] != 0:
                nums[slow], nums[fast] = nums[fast], nums[slow]
                slow += 1
```

## 4. Next Permutation

**题型识别**

```text
排列 / Permutation Construction
```

**中文解释**

从右往左找第一个下降位置 `i`，也就是 `nums[i] < nums[i+1]`。  
然后从右边找第一个比 `nums[i]` 大的数交换。  
最后把 `i+1` 后面的部分反转，得到刚好更大的最小排列。

**English Explanation**

Scan from right to left and find the first index `i` such that `nums[i] < nums[i+1]`.  
Then find the first number from the right that is greater than `nums[i]` and swap them.  
Finally, reverse the suffix to get the smallest permutation that is still larger.

**Python 标准解法**

```python
from typing import List


class Solution:
    def nextPermutation(self, nums: List[int]) -> None:
        n = len(nums)
        i = n - 2

        while i >= 0 and nums[i] >= nums[i + 1]:
            i -= 1

        if i >= 0:
            j = n - 1
            while nums[j] <= nums[i]:
                j -= 1
            nums[i], nums[j] = nums[j], nums[i]

        left, right = i + 1, n - 1
        while left < right:
            nums[left], nums[right] = nums[right], nums[left]
            left += 1
            right -= 1
```

## 5. Merge Sorted Array

**题型识别**

```text
双指针，从后往前 / Two Pointers from the End
```

**中文解释**

因为 `nums1` 前面有有效元素，如果从前往后合并会覆盖。  
所以从后往前填，把较大的元素放到末尾。

**English Explanation**

If we merge from the front, we may overwrite valid elements in `nums1`.  
So we merge from the back and place the larger value at the end each time.

**Python 标准解法**

```python
from typing import List


class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        i = m - 1
        j = n - 1
        k = m + n - 1

        while j >= 0:
            if i >= 0 and nums1[i] > nums2[j]:
                nums1[k] = nums1[i]
                i -= 1
            else:
                nums1[k] = nums2[j]
                j -= 1
            k -= 1
```

## 6. Sort Colors

**题型识别**

```text
三指针 / Dutch National Flag
```

**中文解释**

维护三个区域：

- 左边全是 `0`
- 右边全是 `2`
- 中间尚未处理

遍历时把当前元素放到正确区域。

**English Explanation**

Maintain three regions:

- all `0`s on the left
- all `2`s on the right
- the middle is unprocessed

We place the current number into the correct region as we scan.

**Python 标准解法**

```python
from typing import List


class Solution:
    def sortColors(self, nums: List[int]) -> None:
        left = 0
        i = 0
        right = len(nums) - 1

        while i <= right:
            if nums[i] == 0:
                nums[left], nums[i] = nums[i], nums[left]
                left += 1
                i += 1
            elif nums[i] == 2:
                nums[right], nums[i] = nums[i], nums[right]
                right -= 1
            else:
                i += 1
```

## 7. Add Strings

**题型识别**

```text
模拟加法 / String Simulation
```

**中文解释**

从后往前逐位相加，维护进位 `carry`。  
和链表版 `Add Two Numbers` 本质一样，只不过输入是字符串。

**English Explanation**

Add digits from right to left and maintain a carry.  
This is the same idea as `Add Two Numbers`, except the input is a string.

**Python 标准解法**

```python
class Solution:
    def addStrings(self, num1: str, num2: str) -> str:
        i = len(num1) - 1
        j = len(num2) - 1
        carry = 0
        ans = []

        while i >= 0 or j >= 0 or carry:
            x = ord(num1[i]) - ord("0") if i >= 0 else 0
            y = ord(num2[j]) - ord("0") if j >= 0 else 0

            total = x + y + carry
            ans.append(str(total % 10))
            carry = total // 10

            i -= 1
            j -= 1

        return "".join(reversed(ans))
```

## 8. String Compression

**题型识别**

```text
双指针 / Run-Length Encoding
```

**中文解释**

用一个指针找连续相同字符的区间，另一个写指针把压缩结果写回原数组。  
如果某字符连续出现次数大于 1，就把次数拆成数字字符写进去。

**English Explanation**

Use one pointer to find a block of consecutive equal characters, and another write pointer to store the compressed result in-place.  
If a character appears more than once, write the count as digits.

**Python 标准解法**

```python
from typing import List


class Solution:
    def compress(self, chars: List[str]) -> int:
        n = len(chars)
        write = 0
        left = 0

        while left < n:
            right = left
            while right < n and chars[right] == chars[left]:
                right += 1

            chars[write] = chars[left]
            write += 1

            count = right - left
            if count > 1:
                for ch in str(count):
                    chars[write] = ch
                    write += 1

            left = right

        return write
```

## 9. Reverse Integer

**题型识别**

```text
数学模拟 / Number Manipulation
```

**中文解释**

不断取末位数字，拼到结果后面。  
最后检查是否超出 32 位有符号整数范围。

**English Explanation**

Repeatedly take the last digit and append it to the reversed result.  
At the end, check whether it overflows the 32-bit signed integer range.

**Python 标准解法**

```python
class Solution:
    def reverse(self, x: int) -> int:
        sign = -1 if x < 0 else 1
        x = abs(x)

        ans = 0
        while x > 0:
            ans = ans * 10 + x % 10
            x //= 10

        ans *= sign
        if ans < -2**31 or ans > 2**31 - 1:
            return 0
        return ans
```

## 11. Group Anagrams

**题型识别**

```text
哈希分组 / Hash Grouping
```

**中文解释**

异位词排序后一定相同。  
把排序后的字符串当作 key，把原字符串加入对应分组。

**English Explanation**

Anagrams become identical after sorting.  
Use the sorted string as the key and group original strings under that key.

**Python 标准解法**

```python
from collections import defaultdict
from typing import List


class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        groups = defaultdict(list)

        for s in strs:
            key = "".join(sorted(s))
            groups[key].append(s)

        return list(groups.values())
```

## 13. Intersection of Two Arrays II

**题型识别**

```text
计数哈希 / Counter
```

**中文解释**

这次交集要保留重复次数。  
所以统计一个数组中每个数字的出现次数，再扫描另一个数组，能匹配就加入答案并减少计数。

**English Explanation**

Now duplicates matter, so we need counts.  
Count each number in one array, then scan the other array and match while decreasing the count.

**Python 标准解法**

```python
from collections import Counter
from typing import List


class Solution:
    def intersect(self, nums1: List[int], nums2: List[int]) -> List[int]:
        count = Counter(nums1)
        ans = []

        for x in nums2:
            if count[x] > 0:
                ans.append(x)
                count[x] -= 1

        return ans
```

## 14. Find All Duplicates in an Array

**题型识别**

```text
下标映射 / Sign Marking
```

**中文解释**

数组元素范围在 `1..n`，所以数字 `x` 可以映射到下标 `x-1`。  
如果访问这个位置时已经是负数，说明这个数字之前见过一次了。

**English Explanation**

Values are in the range `1..n`, so each number `x` maps to index `x - 1`.  
If the value at that index is already negative, then `x` has appeared before.

**Python 标准解法**

```python
from typing import List


class Solution:
    def findDuplicates(self, nums: List[int]) -> List[int]:
        ans = []

        for x in nums:
            idx = abs(x) - 1
            if nums[idx] < 0:
                ans.append(abs(x))
            else:
                nums[idx] *= -1

        return ans
```

## 15. Find the Duplicate Number

**题型识别**

```text
Floyd 判环 / Cycle Detection
```

**中文解释**

把数组看成链表：

- 下标是节点
- `nums[i]` 是 next 指针

因为有一个重复值，所以一定会形成环。  
先找到相遇点，再找环入口，入口值就是重复数。

**English Explanation**

Treat the array as a linked list:

- the index is a node
- `nums[i]` is the next pointer

Because one value is duplicated, a cycle must exist.  
Find the meeting point first, then find the cycle entrance, which is the duplicate number.

**Python 标准解法**

```python
from typing import List


class Solution:
    def findDuplicate(self, nums: List[int]) -> int:
        slow = nums[0]
        fast = nums[0]

        while True:
            slow = nums[slow]
            fast = nums[nums[fast]]
            if slow == fast:
                break

        slow = nums[0]
        while slow != fast:
            slow = nums[slow]
            fast = nums[fast]

        return slow
```

## 16. Single Number

**题型识别**

```text
位运算 / XOR
```

**中文解释**

相同的数异或两次会抵消：

- `a ^ a = 0`
- `a ^ 0 = a`

所以把所有数异或起来，最后留下的就是只出现一次的数。

**English Explanation**

XOR cancels duplicates:

- `a ^ a = 0`
- `a ^ 0 = a`

So XOR all numbers together and the remaining value is the one that appears only once.

**Python 标准解法**

```python
from typing import List


class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        ans = 0
        for x in nums:
            ans ^= x
        return ans
```

## 17. 3Sum

**题型识别**

```text
排序 + 固定一个数 + 双指针
Sort + Fix One + Two Pointers
```

**中文解释**

先排序。  
枚举第一个数 `nums[i]`，然后在右边区间用双指针找两数之和等于 `-nums[i]`。  
为了避免重复答案，要对 `i`、`left`、`right` 都做去重处理。

**English Explanation**

Sort first.  
Fix one number `nums[i]`, then use two pointers on the remaining suffix to find two numbers that sum to `-nums[i]`.  
To avoid duplicate triplets, we must skip duplicates carefully.

**Python 标准解法**

```python
from typing import List


class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        nums.sort()
        ans = []
        n = len(nums)

        for i in range(n):
            if i > 0 and nums[i] == nums[i - 1]:
                continue

            left, right = i + 1, n - 1
            while left < right:
                total = nums[i] + nums[left] + nums[right]

                if total == 0:
                    ans.append([nums[i], nums[left], nums[right]])
                    left += 1
                    right -= 1

                    while left < right and nums[left] == nums[left - 1]:
                        left += 1
                    while left < right and nums[right] == nums[right + 1]:
                        right -= 1
                elif total < 0:
                    left += 1
                else:
                    right -= 1

        return ans
```

## 18. 3Sum Closest

**题型识别**

```text
排序 + 双指针 / Sort + Two Pointers
```

**中文解释**

和 `3Sum` 很像，只是不再收集所有答案，而是维护一个最接近 `target` 的和。  
根据当前和与目标的大小关系移动 `left` 或 `right`。

**English Explanation**

This is very similar to `3Sum`, except we do not collect all triplets.  
Instead, we keep track of the sum closest to `target` and move pointers based on whether the current sum is too small or too large.

**Python 标准解法**

```python
from typing import List


class Solution:
    def threeSumClosest(self, nums: List[int], target: int) -> int:
        nums.sort()
        n = len(nums)
        best = nums[0] + nums[1] + nums[2]

        for i in range(n - 2):
            left, right = i + 1, n - 1

            while left < right:
                total = nums[i] + nums[left] + nums[right]

                if abs(total - target) < abs(best - target):
                    best = total

                if total < target:
                    left += 1
                elif total > target:
                    right -= 1
                else:
                    return target

        return best
```

## 19. Container With Most Water

**题型识别**

```text
左右双指针 / Two Pointers
```

**中文解释**

面积由宽度和较短的板决定。  
如果不移动短板，宽度只会变小，而短板不变，面积不可能更优。  
所以每次移动较短的一边。

**English Explanation**

The area is determined by width and the shorter wall.  
If we keep the shorter wall, width only shrinks and the limiting height does not improve, so the area cannot get better.  
Therefore, we always move the shorter side.

**Python 标准解法**

```python
from typing import List


class Solution:
    def maxArea(self, height: List[int]) -> int:
        left, right = 0, len(height) - 1
        ans = 0

        while left < right:
            h = min(height[left], height[right])
            ans = max(ans, h * (right - left))

            if height[left] < height[right]:
                left += 1
            else:
                right -= 1

        return ans
```

## 20. Squares of a Sorted Array

**题型识别**

```text
双指针 / Two Pointers
```

**中文解释**

平方后最大的数一定来自最左边的负数绝对值或者最右边的正数。  
所以用左右双指针比较绝对值，把较大的平方从后往前填入答案。

**English Explanation**

After squaring, the largest value must come from either the leftmost negative number or the rightmost positive number.  
Use two pointers and fill the answer array from right to left.

**Python 标准解法**

```python
from typing import List


class Solution:
    def sortedSquares(self, nums: List[int]) -> List[int]:
        n = len(nums)
        ans = [0] * n
        left, right = 0, n - 1
        idx = n - 1

        while left <= right:
            if abs(nums[left]) > abs(nums[right]):
                ans[idx] = nums[left] * nums[left]
                left += 1
            else:
                ans[idx] = nums[right] * nums[right]
                right -= 1
            idx -= 1

        return ans
```

## 23. Longest Substring Without Repeating Characters

**题型识别**

```text
可变滑动窗口 / Variable Sliding Window
```

**中文解释**

维护一个无重复字符的窗口。  
当右边加入一个重复字符时，就不断移动左边界直到窗口重新合法。

**English Explanation**

Maintain a window with no repeated characters.  
When adding the rightmost character causes a duplicate, move the left boundary until the window becomes valid again.

**Python 标准解法**

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        seen = set()
        left = 0
        ans = 0

        for right, ch in enumerate(s):
            while ch in seen:
                seen.remove(s[left])
                left += 1
            seen.add(ch)
            ans = max(ans, right - left + 1)

        return ans
```

## 24. Minimum Window Substring

**题型识别**

```text
可变滑窗 + 频率统计
Variable Sliding Window + Frequency Count
```

**中文解释**

先扩张窗口，直到它已经覆盖了 `t` 中所有需要的字符。  
然后尽量收缩左边界，找到最短合法窗口。  
核心是维护：

- `need`: 目标频率
- `window`: 当前窗口频率
- `have`: 当前已经满足的字符种类数

**English Explanation**

Expand the window until it covers all required characters from `t`.  
Then shrink from the left as much as possible while keeping the window valid.  
The key is to maintain:

- `need`: target frequencies
- `window`: current window frequencies
- `have`: how many required character groups are already satisfied

**Python 标准解法**

```python
from collections import Counter


class Solution:
    def minWindow(self, s: str, t: str) -> str:
        if not s or not t:
            return ""

        need = Counter(t)
        window = {}
        have = 0
        need_count = len(need)

        res = [-1, -1]
        res_len = float("inf")
        left = 0

        for right, ch in enumerate(s):
            window[ch] = window.get(ch, 0) + 1

            if ch in need and window[ch] == need[ch]:
                have += 1

            while have == need_count:
                if right - left + 1 < res_len:
                    res = [left, right]
                    res_len = right - left + 1

                left_char = s[left]
                window[left_char] -= 1
                if left_char in need and window[left_char] < need[left_char]:
                    have -= 1
                left += 1

        left, right = res
        return s[left:right + 1] if res_len != float("inf") else ""
```

## 25. Longest Substring with At Most K Distinct Characters

**题型识别**

```text
可变滑窗 / Variable Sliding Window
```

**中文解释**

窗口中不同字符种类数不能超过 `k`。  
如果超过，就不断缩左边界，直到重新满足条件。

**English Explanation**

The number of distinct characters inside the window cannot exceed `k`.  
If it does, keep shrinking the left side until the condition becomes valid again.

**Python 标准解法**

```python
class Solution:
    def lengthOfLongestSubstringKDistinct(self, s: str, k: int) -> int:
        if k == 0:
            return 0

        count = {}
        left = 0
        ans = 0

        for right, ch in enumerate(s):
            count[ch] = count.get(ch, 0) + 1

            while len(count) > k:
                left_char = s[left]
                count[left_char] -= 1
                if count[left_char] == 0:
                    del count[left_char]
                left += 1

            ans = max(ans, right - left + 1)

        return ans
```

## 26. Max Consecutive Ones III

**题型识别**

```text
可变滑窗 / Variable Sliding Window
```

**中文解释**

允许最多把 `k` 个 `0` 翻成 `1`。  
所以窗口里 `0` 的数量最多只能有 `k` 个。  
一旦超过，就缩左边界。

**English Explanation**

We may flip at most `k` zeros into ones.  
So the number of zeros inside the window can never exceed `k`.  
Once it does, shrink the window from the left.

**Python 标准解法**

```python
from typing import List


class Solution:
    def longestOnes(self, nums: List[int], k: int) -> int:
        left = 0
        zeros = 0
        ans = 0

        for right, x in enumerate(nums):
            if x == 0:
                zeros += 1

            while zeros > k:
                if nums[left] == 0:
                    zeros -= 1
                left += 1

            ans = max(ans, right - left + 1)

        return ans
```

## 27. Permutation in String

**题型识别**

```text
固定窗口 + 频率比较
Fixed Sliding Window + Frequency Count
```

**中文解释**

如果 `s2` 中某个长度为 `len(s1)` 的子串是 `s1` 的排列，那么这两个字符串的字符频率必然完全相同。  
所以维护固定窗口频率并比较即可。

**English Explanation**

If a substring of `s2` with length `len(s1)` is a permutation of `s1`, then both strings must have exactly the same character frequencies.  
So we keep a fixed-size sliding window and compare counts.

**Python 标准解法**

```python
from collections import Counter


class Solution:
    def checkInclusion(self, s1: str, s2: str) -> bool:
        n1, n2 = len(s1), len(s2)
        if n1 > n2:
            return False

        need = Counter(s1)
        window = Counter(s2[:n1])

        if window == need:
            return True

        for i in range(n1, n2):
            window[s2[i]] += 1
            left_char = s2[i - n1]
            window[left_char] -= 1
            if window[left_char] == 0:
                del window[left_char]

            if window == need:
                return True

        return False
```

## 28. Sliding Window Maximum

**题型识别**

```text
单调队列 / Monotonic Deque
```

**中文解释**

队列中存下标，并保持对应值单调递减。  
这样队首始终是当前窗口最大值的下标。  
每次移动窗口时：

- 移除过期下标
- 把所有比当前值小的尾部下标弹出

**English Explanation**

Store indices in a deque and keep their values in decreasing order.  
This guarantees that the front of the deque is always the maximum of the current window.  
When the window moves:

- remove expired indices
- pop all smaller values from the back

**Python 标准解法**

```python
from collections import deque
from typing import List


class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        dq = deque()
        ans = []

        for i, x in enumerate(nums):
            while dq and dq[0] <= i - k:
                dq.popleft()

            while dq and nums[dq[-1]] <= x:
                dq.pop()

            dq.append(i)

            if i >= k - 1:
                ans.append(nums[dq[0]])

        return ans
```

## 这一组最后怎么总结

**中文**

这一组题虽然题目很多，但模板非常集中：

- `Two Sum / Group Anagrams / Valid Anagram / Intersection` 属于哈希
- `Move Zeroes / Merge Sorted Array / Sort Colors` 属于原地双指针
- `3Sum / 3Sum Closest / Container / Squares` 属于排序后双指针
- `Longest Substring / Minimum Window / K Distinct / Max Ones III` 属于滑窗
- `Sliding Window Maximum` 属于滑窗 + 单调队列

**English**

Although this category contains many problems, the actual templates are very concentrated:

- `Two Sum / Group Anagrams / Valid Anagram / Intersection` are hashing problems
- `Move Zeroes / Merge Sorted Array / Sort Colors` are in-place two-pointer problems
- `3Sum / 3Sum Closest / Container / Squares` are sorted two-pointer problems
- `Longest Substring / Minimum Window / K Distinct / Max Ones III` are sliding window problems
- `Sliding Window Maximum` is sliding window + monotonic deque
