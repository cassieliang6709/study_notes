# 08 动态规划完整答案

说明：

- 本文按题目分节
- 每题包含：题型识别、中文解释、English explanation、Python 标准解法
- DP 最重要的不是代码，而是先定义清楚 `dp[i]` 或 `dp[i][j]` 的含义

## 1. Climbing Stairs

**题型识别**

```text
一维 DP / 1D DP
```

**中文解释**

到第 `i` 阶的方法数 = 到 `i-1` 阶的方法数 + 到 `i-2` 阶的方法数。  
这是最基础的 Fibonacci 型 DP。

**English Explanation**

The number of ways to reach step `i` equals:

```text
ways[i - 1] + ways[i - 2]
```

This is the most basic Fibonacci-style DP.

**Python 标准解法**

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2:
            return n

        a, b = 1, 2
        for _ in range(3, n + 1):
            a, b = b, a + b

        return b
```

## 2. Fibonacci Number

**题型识别**

```text
基础递推 / Basic Recurrence
```

**中文解释**

这是最原始的 DP 递推题：

```text
fib(n) = fib(n-1) + fib(n-2)
```

**English Explanation**

This is the original DP recurrence:

```text
fib(n) = fib(n-1) + fib(n-2)
```

**Python 标准解法**

```python
class Solution:
    def fib(self, n: int) -> int:
        if n < 2:
            return n

        a, b = 0, 1
        for _ in range(2, n + 1):
            a, b = b, a + b

        return b
```

## 3. Best Time to Buy and Sell Stock

**题型识别**

```text
状态维护 / Running DP State
```

**中文解释**

虽然这题常用贪心写，但也可以理解成 DP。  
核心状态是“到当前位置为止的最低买入价”，然后不断更新最大利润。

**English Explanation**

Although this problem is often solved greedily, it can also be viewed as DP.  
The key state is the minimum buying price seen so far, and we update the maximum profit along the scan.

**Python 标准解法**

```python
from typing import List


class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        min_price = float("inf")
        ans = 0

        for p in prices:
            min_price = min(min_price, p)
            ans = max(ans, p - min_price)

        return ans
```

## 4. Longest Increasing Subsequence

**题型识别**

```text
一维 DP / 1D DP
```

**中文解释**

定义：

```text
dp[i] = 以 nums[i] 结尾的最长递增子序列长度
```

然后枚举前面的 `j < i`，如果 `nums[j] < nums[i]`，就可以转移：

```text
dp[i] = max(dp[i], dp[j] + 1)
```

**English Explanation**

Define:

```text
dp[i] = length of the longest increasing subsequence ending at nums[i]
```

Then for each previous `j < i`, if `nums[j] < nums[i]`, we may transition:

```text
dp[i] = max(dp[i], dp[j] + 1)
```

**Python 标准解法**

```python
from typing import List


class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        dp = [1] * len(nums)

        for i in range(len(nums)):
            for j in range(i):
                if nums[j] < nums[i]:
                    dp[i] = max(dp[i], dp[j] + 1)

        return max(dp, default=0)
```

## 5. Maximum Product Subarray

**题型识别**

```text
状态双维护 / Track Max and Min
```

**中文解释**

这题不能只维护最大值，因为负数可能把最小积变成最大积。  
所以要同时维护：

- `cur_max`
- `cur_min`

**English Explanation**

We cannot track only the maximum product, because a negative number may turn the minimum product into the maximum.  
So we must track both:

- `cur_max`
- `cur_min`

**Python 标准解法**

```python
from typing import List


class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        cur_max = cur_min = ans = nums[0]

        for x in nums[1:]:
            vals = (x, cur_max * x, cur_min * x)
            cur_max = max(vals)
            cur_min = min(vals)
            ans = max(ans, cur_max)

        return ans
```

## 6. Word Break

**题型识别**

```text
前缀可达 DP / Prefix Feasibility DP
```

**中文解释**

定义：

```text
dp[i] = s[:i] 能否被拆分
```

如果存在某个 `j < i`，满足：

- `dp[j] == True`
- `s[j:i]` 在词典中

那么 `dp[i] = True`。

**English Explanation**

Define:

```text
dp[i] = whether s[:i] can be segmented
```

If there exists some `j < i` such that:

- `dp[j] == True`
- `s[j:i]` is in the dictionary

then `dp[i] = True`.

**Python 标准解法**

```python
from typing import List


class Solution:
    def wordBreak(self, s: str, wordDict: List[str]) -> bool:
        word_set = set(wordDict)
        dp = [False] * (len(s) + 1)
        dp[0] = True

        for i in range(1, len(s) + 1):
            for j in range(i):
                if dp[j] and s[j:i] in word_set:
                    dp[i] = True
                    break

        return dp[-1]
```

## 7. Decode Ways

**题型识别**

```text
字符串一维 DP / String 1D DP
```

**中文解释**

每个位置都可以考虑：

- 单独解码一位
- 或和前一位组合成两位数解码

所以 `dp[i]` 由：

- `dp[i-1]`
- `dp[i-2]`

转移而来。

**English Explanation**

At each position, we can consider:

- decoding one digit alone
- or decoding two digits together with the previous one

So `dp[i]` comes from:

- `dp[i-1]`
- `dp[i-2]`

**Python 标准解法**

```python
class Solution:
    def numDecodings(self, s: str) -> int:
        if s[0] == "0":
            return 0

        dp = [0] * (len(s) + 1)
        dp[0] = dp[1] = 1

        for i in range(2, len(s) + 1):
            if s[i - 1] != "0":
                dp[i] += dp[i - 1]
            if "10" <= s[i - 2:i] <= "26":
                dp[i] += dp[i - 2]

        return dp[-1]
```

## 8. Longest Palindromic Subsequence

**题型识别**

```text
区间 DP / Interval DP
```

**中文解释**

定义：

```text
dp[i][j] = s[i..j] 的最长回文子序列长度
```

如果 `s[i] == s[j]`，就能把两端一起用上；否则取两边去掉一个字符后的较大值。

**English Explanation**

Define:

```text
dp[i][j] = length of the longest palindromic subsequence in s[i..j]
```

If `s[i] == s[j]`, we can use both ends. Otherwise, take the better result after removing one side.

**Python 标准解法**

```python
class Solution:
    def longestPalindromeSubseq(self, s: str) -> int:
        n = len(s)
        dp = [[0] * n for _ in range(n)]

        for i in range(n - 1, -1, -1):
            dp[i][i] = 1
            for j in range(i + 1, n):
                if s[i] == s[j]:
                    dp[i][j] = dp[i + 1][j - 1] + 2
                else:
                    dp[i][j] = max(dp[i + 1][j], dp[i][j - 1])

        return dp[0][n - 1]
```

## 9. Maximum Subarray

**题型识别**

```text
Kadane / Ending-at-i DP
```

**中文解释**

定义：

```text
cur = 以当前元素结尾的最大子数组和
```

那么每一步：

- 要么从当前元素重新开始
- 要么接在前一个最优结尾后面

**English Explanation**

Define:

```text
cur = maximum subarray sum ending at the current index
```

At each step:

- either start fresh from the current value
- or extend the previous best ending here

**Python 标准解法**

```python
from typing import List


class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        cur = ans = nums[0]

        for x in nums[1:]:
            cur = max(x, cur + x)
            ans = max(ans, cur)

        return ans
```

## 10. Unique Paths

**题型识别**

```text
网格 DP / Grid DP
```

**中文解释**

到某个格子的路径数 = 上边来的路径数 + 左边来的路径数。  
因为只能向右或向下走。

**English Explanation**

The number of ways to reach a cell equals:

```text
ways from above + ways from the left
```

because we can only move right or down.

**Python 标准解法**

```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [1] * n

        for _ in range(1, m):
            for j in range(1, n):
                dp[j] += dp[j - 1]

        return dp[-1]
```

## 11. Unique Paths II

**题型识别**

```text
带障碍网格 DP / Grid DP with Obstacles
```

**中文解释**

和 `Unique Paths` 一样，但遇到障碍时该位置路径数为 `0`。  
可以继续用一维 DP 压缩空间。

**English Explanation**

This is the same as `Unique Paths`, except blocked cells contribute `0` paths.  
We can still use 1D DP for space optimization.

**Python 标准解法**

```python
from typing import List


class Solution:
    def uniquePathsWithObstacles(self, obstacleGrid: List[List[int]]) -> int:
        n = len(obstacleGrid[0])
        dp = [0] * n
        dp[0] = 1

        for row in obstacleGrid:
            for j in range(n):
                if row[j] == 1:
                    dp[j] = 0
                elif j > 0:
                    dp[j] += dp[j - 1]

        return dp[-1]
```

## 12. Subarray Sum Equals K

**题型识别**

```text
前缀和 + 哈希 / Prefix Sum + Hash Map
```

**中文解释**

虽然这题常被归在 DP 一组里，但本质不是传统 DP，而是前缀和。  
如果当前前缀和是 `prefix`，那么只要之前有前缀和 `prefix - k`，中间这段子数组和就是 `k`。

**English Explanation**

Although this is often grouped with DP problems, the core idea is prefix sums.  
If the current prefix sum is `prefix`, then any earlier prefix sum equal to `prefix - k` forms a subarray summing to `k`.

**Python 标准解法**

```python
from collections import defaultdict
from typing import List


class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        count = defaultdict(int)
        count[0] = 1
        prefix = 0
        ans = 0

        for x in nums:
            prefix += x
            ans += count[prefix - k]
            count[prefix] += 1

        return ans
```

## 13. Continuous Subarray Sum

**题型识别**

```text
前缀和取模 / Prefix Sum Modulo
```

**中文解释**

如果两个前缀和对 `k` 取模结果相同，那么它们之间的子数组和一定是 `k` 的倍数。  
所以记录“某个余数第一次出现的位置”，只要间隔超过 1 就满足题意。

**English Explanation**

If two prefix sums have the same remainder modulo `k`, then the subarray between them sums to a multiple of `k`.  
So we record the first index where each remainder appears, and if the distance is greater than 1, the condition is satisfied.

**Python 标准解法**

```python
from typing import List


class Solution:
    def checkSubarraySum(self, nums: List[int], k: int) -> bool:
        pos = {0: -1}
        total = 0

        for i, x in enumerate(nums):
            total += x
            rem = total % k if k != 0 else total
            if rem in pos:
                if i - pos[rem] > 1:
                    return True
            else:
                pos[rem] = i

        return False
```

## 14. Longest Palindromic Substring

**题型识别**

```text
中心扩展 / Expand Around Center
```

**中文解释**

虽然也能用区间 DP，但最适合面试手写的是中心扩展。  
每个回文串都有一个中心：

- 奇数长度：单个字符
- 偶数长度：两个字符之间

从中心向两边扩展即可。

**English Explanation**

Although interval DP also works, the most interview-friendly solution is expand-around-center.  
Every palindrome has a center:

- odd length: one character
- even length: between two characters

Expand outward from every possible center.

**Python 标准解法**

```python
class Solution:
    def longestPalindrome(self, s: str) -> str:
        ans = ""

        def expand(left, right):
            while left >= 0 and right < len(s) and s[left] == s[right]:
                left -= 1
                right += 1
            return s[left + 1:right]

        for i in range(len(s)):
            s1 = expand(i, i)
            s2 = expand(i, i + 1)
            ans = max(ans, s1, s2, key=len)

        return ans
```

## 15. Regular Expression Matching

**题型识别**

```text
二维匹配 DP / 2D Matching DP
```

**中文解释**

定义：

```text
dp[i][j] = s[:i] 是否能匹配 p[:j]
```

处理两种关键情况：

- 普通字符或 `.`：看左上角
- `*`：要么表示前一个字符出现 0 次，要么表示出现多次

**English Explanation**

Define:

```text
dp[i][j] = whether s[:i] matches p[:j]
```

There are two key cases:

- normal character or `.`: look at the diagonal
- `*`: either use zero occurrences of the previous token, or use one/more occurrences

**Python 标准解法**

```python
class Solution:
    def isMatch(self, s: str, p: str) -> bool:
        m, n = len(s), len(p)
        dp = [[False] * (n + 1) for _ in range(m + 1)]
        dp[0][0] = True

        for j in range(2, n + 1):
            if p[j - 1] == "*":
                dp[0][j] = dp[0][j - 2]

        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if p[j - 1] in {s[i - 1], "."}:
                    dp[i][j] = dp[i - 1][j - 1]
                elif p[j - 1] == "*":
                    dp[i][j] = dp[i][j - 2]
                    if p[j - 2] in {s[i - 1], "."}:
                        dp[i][j] |= dp[i - 1][j]

        return dp[m][n]
```

## 16. Wildcard Matching

**题型识别**

```text
二维通配 DP / 2D Wildcard DP
```

**中文解释**

定义同样是：

```text
dp[i][j] = s[:i] 是否匹配 p[:j]
```

区别在于：

- `?` 匹配一个任意字符
- `*` 匹配任意长度字符串

所以：

- `?` 看左上角
- `*` 可以来自左边或上边

**English Explanation**

The definition is the same:

```text
dp[i][j] = whether s[:i] matches p[:j]
```

The difference is:

- `?` matches exactly one arbitrary character
- `*` matches any sequence of characters

So:

- `?` depends on the diagonal
- `*` can come from the left or from above

**Python 标准解法**

```python
class Solution:
    def isMatch(self, s: str, p: str) -> bool:
        m, n = len(s), len(p)
        dp = [[False] * (n + 1) for _ in range(m + 1)]
        dp[0][0] = True

        for j in range(1, n + 1):
            if p[j - 1] == "*":
                dp[0][j] = dp[0][j - 1]

        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if p[j - 1] == "*":
                    dp[i][j] = dp[i][j - 1] or dp[i - 1][j]
                elif p[j - 1] == "?" or p[j - 1] == s[i - 1]:
                    dp[i][j] = dp[i - 1][j - 1]

        return dp[m][n]
```

## 17. Longest Increasing Path in a Matrix

**题型识别**

```text
DFS + 记忆化 / DFS + Memoization
```

**中文解释**

每个格子都可以看成一个状态。  
从这个格子出发，最长递增路径长度只和更大的相邻格子有关。  
所以用 DFS 搜索，并用记忆化缓存每个格子的答案。

**English Explanation**

Each cell can be treated as a state.  
The longest increasing path starting from that cell depends only on its larger neighbors.  
So we run DFS and memoize the result for each cell.

**Python 标准解法**

```python
from functools import lru_cache
from typing import List


class Solution:
    def longestIncreasingPath(self, matrix: List[List[int]]) -> int:
        m, n = len(matrix), len(matrix[0])

        @lru_cache(None)
        def dfs(r, c):
            ans = 1
            for dr, dc in ((1, 0), (-1, 0), (0, 1), (0, -1)):
                nr, nc = r + dr, c + dc
                if 0 <= nr < m and 0 <= nc < n and matrix[nr][nc] > matrix[r][c]:
                    ans = max(ans, 1 + dfs(nr, nc))
            return ans

        return max(dfs(r, c) for r in range(m) for c in range(n))
```
