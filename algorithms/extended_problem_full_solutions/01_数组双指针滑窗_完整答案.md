
**这个题型 / 算法点的总结**

这题最重要的不是记答案，而是识别信号。看到 `Two Sum` 时，优先把它归到 `哈希表 / Hash Map` 这类模板，再去套对应的不变量、状态定义或数据结构。

**题目含义**

题目问的是：是否存在两个数之和等于 `target`。  
扫到当前数字 `x` 时，只需要知道之前有没有出现过 `target - x`。  
所以用字典记录“某个值第一次出现的位置”。

**Python 代码**

```python
from typing import List

class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        pos = {}

        for i, x in enumerate(nums):
            need = target - x
            if need in pos:
                return [pos[need], i]
            pos[x] = i

        return []
```

**时间复杂度**

`O(n)`。

**空间复杂度**

`O(n)`。

## 2. Product of Array Except Self

**题型识别**

```text
前缀乘积 + 后缀乘积 / Prefix and Suffix Products
```

**这个题型 / 算法点的总结**

这题最重要的不是记答案，而是识别信号。看到 `Product of Array Except Self` 时，优先把它归到 `前缀乘积 + 后缀乘积 / Prefix and Suffix Products` 这类模板，再去套对应的不变量、状态定义或数据结构。

**题目含义**

`answer[i]` 要等于 `i` 左边所有数的乘积乘上右边所有数的乘积。  
第一遍从左到右，把左边乘积存进 `answer`。  
第二遍从右到左，再乘上右边乘积。

**Python 代码**

```python
from typing import List

class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        n = len(nums)
        ans = [1] * n

        left = 1
        for i in range(n):
            ans[i] = left
            left *= nums[i]

        right = 1
        for i in range(n - 1, -1, -1):
            ans[i] *= right
            right *= nums[i]

        return ans
```

**时间复杂度**

`O(n)`。

**空间复杂度**

`O(1)` 额外空间；如果把输出数组算上则是 `O(n)`。

先看题型识别里的信号：这题本质上就是 `前缀乘积 + 后缀乘积 / Prefix and Suffix Products`。把题目翻译成这个模板后，再去套对应的不变量、状态定义或数据结构，就会更容易写出来。

**示例 case**

- 输入：`nums = [1,2,3,4]`
- 输出：`[24,12,8,6]`。每个位置都等于左边乘积乘右边乘积。

**题目含义**

`slow` 表示下一个非零元素应该放的位置，`fast` 负责扫描数组。  
每当 `fast` 遇到非零元素，就把它交换到 `slow` 位置。

**Python 代码**

```python
from typing import List

class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        slow = 0

        for fast in range(len(nums)):
            if nums[fast] != 0:
                nums[slow], nums[fast] = nums[fast], nums[slow]
                slow += 1
```

`O(n)`。

## 4. Next Permutation

**题型识别**

```text

```

**这个题型 / 算法点的总结**

这题最重要的不是记答案，而是识别信号。看到 `Next Permutation` 时，优先把它归到 `排列 / Permutation Construction` 这类模板，再去套对应的不变量、状态定义或数据结构。

**题目含义**

从右往左找第一个下降位置 `i`，也就是 `nums[i] < nums[i+1]`。  
然后从右边找第一个比 `nums[i]` 大的数交换。  
最后把 `i+1` 后面的部分反转，得到刚好更大的最小排列。

**Python 代码**

```python
from typing import List

class Solution:
    def nextPermutation(self, nums: List[int]) -> None:
        n = len(nums)
        i = n - 2

        while i >= 0 and nums[i] >= nums[i + 1]:
            i -= 1

        if i >= 0:
            j = n - 1
            while nums[j] <= nums[i]:
                j -= 1
            nums[i], nums[j] = nums[j], nums[i]

        left, right = i + 1, n - 1
        while left < right:
            nums[left], nums[right] = nums[right], nums[left]
            left += 1
            right -= 1
```

**时间复杂度**

`O(n)`。

**空间复杂度**

`O(1)`。

**题目含义**

因为 `nums1` 前面有有效元素，如果从前往后合并会覆盖。  
所以从后往前填，把较大的元素放到末尾。

**Python 代码**

```python
from typing import List

class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        i = m - 1
        j = n - 1
        k = m + n - 1

        while j >= 0:
            if i >= 0 and nums1[i] > nums2[j]:
                nums1[k] = nums1[i]
                i -= 1
            else:
                nums1[k] = nums2[j]
                j -= 1
            k -= 1
```

`O(m+n)`。

**题目含义**

从后往前逐位相加，维护进位 `carry`。  
和链表版 `Add Two Numbers` 本质一样，只不过输入是字符串。

**Python 代码**

```python
class Solution:
    def addStrings(self, num1: str, num2: str) -> str:
        i = len(num1) - 1
        j = len(num2) - 1
        carry = 0
        ans = []

        while i >= 0 or j >= 0 or carry:
            x = ord(num1[i]) - ord("0") if i >= 0 else 0
            y = ord(num2[j]) - ord("0") if j >= 0 else 0

            total = x + y + carry
            ans.append(str(total % 10))
            carry = total // 10

            i -= 1
            j -= 1

        return "".join(reversed(ans))
```

**题目含义**

用一个指针找连续相同字符的区间，另一个写指针把压缩结果写回原数组。  
如果某字符连续出现次数大于 1，就把次数拆成数字字符写进去。

**Python 代码**

```python
from typing import List

class Solution:
    def compress(self, chars: List[str]) -> int:
        n = len(chars)
        write = 0
        left = 0

        while left < n:
            right = left
            while right < n and chars[right] == chars[left]:
                right += 1

            chars[write] = chars[left]
            write += 1

            count = right - left
            if count > 1:
                for ch in str(count):
                    chars[write] = ch
                    write += 1

            left = right

        return write
```

通常是 `O(n)`。

```python
class Solution:
    def reverse(self, x: int) -> int:

        x = abs(x)

        ans = 0
        while x > 0:
            ans = ans * 10 + x % 10
            x //= 10

        ans *= sign
        if ans < -2**31 or ans > 2**31 - 1:
            return 0
        return ans
```

```python
from collections import defaultdict
from typing import List

class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        groups = defaultdict(list)

        for s in strs:
            key = "".join(sorted(s))
            groups[key].append(s)

        return list(groups.values())
```

**Python 代码**

```python
from typing import List

class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        nums.sort()
        ans = []
        n = len(nums)

        for i in range(n):
            if i > 0 and nums[i] == nums[i - 1]:
                continue

            left, right = i + 1, n - 1
            while left < right:
                total = nums[i] + nums[left] + nums[right]

                if total == 0:
                    ans.append([nums[i], nums[left], nums[right]])
                    left += 1
                    right -= 1

                    while left < right and nums[left] == nums[left - 1]:
                        left += 1
                    while left < right and nums[right] == nums[right + 1]:
                        right -= 1
                elif total < 0:
                    left += 1
                else:
                    right -= 1

        return ans
```

**题型识别**


**这个题型 / 算法点的总结**

这题最重要的不是记答案，而是识别信号。看到 `Minimum Window Substring` 时，优先把它归到 `可变滑窗 + 频率统计 / Variable Sliding Window + Frequency Count` 这类模板，再去套对应的不变量、状态定义或数据结构。

**题目含义**

先扩张窗口，直到它已经覆盖了 `t` 中所有需要的字符。  
然后尽量收缩左边界，找到最短合法窗口。  
核心是维护：

- `need`: 目标频率
- `window`: 当前窗口频率
- `have`: 当前已经满足的字符种类数

**Python 代码**

```python
from collections import Counter

class Solution:
    def minWindow(self, s: str, t: str) -> str:
        if not s or not t:
            return ""

        need = Counter(t)
        window = {}
        have = 0
        need_count = len(need)

        res = [-1, -1]
        res_len = float("inf")
        left = 0

        for right, ch in enumerate(s):
            window[ch] = window.get(ch, 0) + 1

            if ch in need and window[ch] == need[ch]:
                have += 1

            while have == need_count:
                if right - left + 1 < res_len:
                    res = [left, right]
                    res_len = right - left + 1

                left_char = s[left]
                window[left_char] -= 1
                if left_char in need and window[left_char] < need[left_char]:
                    have -= 1
                left += 1

        left, right = res
        return s[left:right + 1] if res_len != float("inf") else ""
```

`O(m+n)`。

**空间复杂度**

`O(k)`。

```python
class Solution:
    def lengthOfLongestSubstringKDistinct(self, s: str, k: int) -> int:
        if k == 0:
            return 0

        count = {}
        left = 0
        ans = 0

        for right, ch in enumerate(s):
            count[ch] = count.get(ch, 0) + 1

            while len(count) > k:
                left_char = s[left]
                count[left_char] -= 1
                if count[left_char] == 0:
                    del count[left_char]
                left += 1

            ans = max(ans, right - left + 1)

        return ans
```

```python
class Solution:
    def checkInclusion(self, s1: str, s2: str) -> bool:
        n1, n2 = len(s1), len(s2)
        if n1 > n2:
            return False

        need = Counter(s1)

            left_char = s2[i - n1]

            if window[left_char] == 0:
                del window[left_char]

            if window == need:
                return True

        return False
```

## 28. Sliding Window Maximum

**题型识别**

```text
单调队列 / Monotonic Deque
```

**这个题型 / 算法点的总结**

这题最重要的不是记答案，而是识别信号。看到 `Sliding Window Maximum` 时，优先把它归到 `单调队列 / Monotonic Deque` 这类模板，再去套对应的不变量、状态定义或数据结构。

**题目含义**

队列中存下标，并保持对应值单调递减。  
这样队首始终是当前窗口最大值的下标。  
每次移动窗口时：

- 移除过期下标
- 把所有比当前值小的尾部下标弹出

**Python 代码**

```python
from collections import deque
from typing import List

class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        dq = deque()
        ans = []

        for i, x in enumerate(nums):
            while dq and dq[0] <= i - k:
                dq.popleft()

            while dq and nums[dq[-1]] <= x:
                dq.pop()

            dq.append(i)

            if i >= k - 1:
                ans.append(nums[dq[0]])

        return ans
```

`O(n)`。

**空间复杂度**

`O(k)`。
