# Algorithms

这个模块负责 coding interview 学习。它不是平铺题库，而是按“先建地图，再做题，再补答案”的顺序组织的。

## 适合谁

- 刷题经常不知道该往哪个 pattern 上靠的人
- 会看答案，但自己讲不清思路的人
- 想系统补树、图、链表、堆这些高频专题的人

## Beginner 推荐顺序

1. 先读一路线图
2. 再读一个专题讲义
3. 进入 `LeetCode 100 Detailed`
4. 再扩到 `Extended Problems Detailed`
5. 卡住时回 `Full Solutions`
6. 面前冲刺时看 `Company Hot Questions`
